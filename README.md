# Claire_n8n

Automation workflows combining [n8n](https://n8n.io) and [ElevenLabs](https://elevenlabs.io) for AI-powered voice and audio pipelines.

## Structure
- `/n8n_workflows` — n8n exported workflow JSON files
- `/scripts` — Helper scripts and utilities
- `/docs` — Notes and documentation

## Setup
1. Import workflow `.json` files into your n8n instance
2. Configure ElevenLabs API credentials in n8n
3. Set required environment variables (see `.env.example`)

## Requirements
- n8n instance (self-hosted or cloud)
- ElevenLabs API key
